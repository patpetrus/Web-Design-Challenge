Latitude Visualizations

Does latitude impact the weather of certain regions of earth? If so, how? These are the questions that informed my project, undertaken as part of the Butler University Data Analytics 2020/2021 Executive Development Bootcamp. To perform this analysis, I started by using the OpenWeather API database to generate a query of over 700 randomized regions across the earth.

Once I created the dataset, I then used MatPlotLib to create visualizations comparing four weather components--average percentage of cloud cover, average humidity, average temperature, and average wind speed--with a region's latitude.

This site contains those visualizations, as well as the full dataset and all relevant observational findings. Enjoy your stay!
